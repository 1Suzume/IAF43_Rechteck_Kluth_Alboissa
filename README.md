# IAF43\_Rechteck\_Kluth\_Alboissa

Ein Windows Forms projekt mit Saif und Noah.

Im Programm kann man die Höhe und Breite eingaben eines Vierecks. Mit hilfe der Parametern werden Umfang Flächeninhalt und die Diagonale berechnet. Dazu gibt es eine Zoom Funktion mit der man das Viereck pro klick um 10% kleiner oder größer macht. Das Viereck kann man auch VIsualisieren das es gerendert wird.
